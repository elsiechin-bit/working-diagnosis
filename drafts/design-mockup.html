Key Design Changes

1. Sentence Case for Titles

Changed from ALL CAPS to mixed case. Significantly improves readability and scanning speed. Users can recognize word shapes better.

2. Visual Hierarchy

Labels remain uppercase (small text is OK), titles use Title Case, descriptions use sentence case. Creates clear levels of information.

3. Breathing Room

Increased line-height and padding. Single-column layout for better readability. Ochre left border provides visual anchor.

4. Font Sizing

Increased from cramped to spacious. Body text is now comfortable at 0.95rem instead of smaller uppercase.

5. Color Consistency

Maintains your palette: paper, ink, ochre, green. Ochre kickers with left border replace cramped layout.

6. Card Layout

From 2-column crammed grid to full-width cards. Each card is immediately scannable. Border provides visual anchor.
